Cream NG

The complete library of accessible and semantic Angular components

Why use CreamNG ?

If you are looking for a perfect Angular component library you are in the right place NG Cream is a collection for Angular. All these components are open source and free. NG Cream is developed by the Studio within Worldline.

  • HTML 5

    Each component has its own code, it is semantic and accessible

  • open source

    All the components are open source and free to use with GitLab

  • CSS 3

    All the latest CSS3 attributes with compatibility for each browser

  • Share with community

    You can send us your components which will be tested before being put online

  • Responsive Design

    All components are responsive (smartphone, tablet, desktop, ...)

  • Accessibility

    All the components are accessible with Web Content Accessibility Guidelines 2.0 (FR : RGAA)
